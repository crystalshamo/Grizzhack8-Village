
## About

Village is a community-focused mobile app that connects parents locally to share advice, support, and real-life experiences through chats, groups, events, and mentorship connections.

---

## Features

-  Parenting advice boards (posts, replies, images)
-  Social feed with trending posts
-  Local group discovery & event planning
-  Parent matching by child age
-  Real-time chat (group + private messaging)
-  Mentorship between new and experienced parents
-  Daily prompts (wins, struggles, reflections)
-  Connections with parenting organizations

---

##  Tech Stack

- React Native (Expo)
- TypeScript
- Node.js + Express
- MySQL
- REST API
- React Navigation
